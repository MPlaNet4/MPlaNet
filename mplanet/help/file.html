File Sharing

If you want to access files of some user who has logged on the MPlaNet network, select the icon of the user and press the ‘File Sharing’ button on the toolbar. A message will be displayed on the screen of that user telling him that you want to access his files. He may or may not allow you to access his system.

If he allows you to access his system, then you can go anywhere on this system and view and copy any files to your system. But, you cannot modify his system because you have read-only access to all remote systems.

In addition to copying files from his computer, you can search for a particular file or folder in his system. For this, you can specify either the file or folder name or a regular expression representing the file or folder.

Once you have finished copying required files, you can end File Sharing by clicking the ‘File Sharing’ button. The button to start file sharing and stop it is same. But the ‘Disable’ icon at the bottom right corner of the button and the tool tip help you in distinguishing between the starting and stopping file sharing, similar to remote assistance, as given above.

 

The operations supported:

Open Folder

·       Double click on the folder you would like to view.

Open File

·       Double click the file you want to open.

·        If it is a JPEG, GIF or PNG file, it will be opened in the image viewer. Otherwise, it will be opened in the text editor. The text editor supports files up to 1MB.

Copy

·        Select the file you would like to copy.

·        Click on the ‘Copy’ button on the toolbar of the pane of the remote machine.

Paste

·        Open the directory on your system in which you want to put the file and click ‘Paste’.

Search

·        Click on the ‘Search’ button on the toolbar of the remote user in your File Sharing window.

·        In the box given for file name, give the file or folder name to be found or specify the regular expression representing the file or folder name.

·        Press the ‘Start’ button.

·        If you want to stop searching, press the stop button.

·        When search is going on, the ‘Search’ button turns into ‘Hide’ button. That button can be used to finish the search and hide the searching part of the window.
